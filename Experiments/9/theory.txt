activity for api :
A Web API (Application Programming Interface) is a system that allows 
communication between frontend (UI) and backend (server/database) over the internet.
It acts as a bridge:
Frontend (React, Angular, mobile apps)
Backend (.NET, Node.js, Java)
Database (SQL, MongoDB)

What is REST API?
A REST API (Representational State Transfer API) is a type of Web API that:
Uses HTTP methods
Works with JSON format
Follows simple, stateless rules
 In short:
REST API = Standard way to build Web APIs using HTTP

what is our endpoint ?
An endpoint is a URL where API is accessed.
http://localhost:5042/api/student

which http methods we are using ?
| Method | Purpose     | Example        |
| ------    | ----------- | -------------- |
| GET     | Read data   | View students  |
| POST   | Add data     | Add student    |
| PUT     | Update data  | Update student  |
| DELETE | Delete data  | Delete student  |

What gives swagger ?
Swagger shows:Available endpoints
Methods (GET, POST, etc.)
Input parameters
Output response

Flow of WEBAPI with Swagger :
User interacts with UI (button click)
UI sends request to API
Controller receives request
Controller processes logic
Data fetched from database
Response sent back (JSON)
UI displays result
